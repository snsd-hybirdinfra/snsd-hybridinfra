# Reports

This directory contains generated detailed validation outputs for repository quality inspection.

## Report role

Reports in this directory are the detailed output layer for diagnostics and checks. Reviewer-facing summaries are maintained under [validation-reports/](../validation-reports/).

## Typical generated reports

- markdown-link-check.md
- portfolio-health-summary.md
- repository-quality-check.md
- repository-summary-report.txt
- root-readme-alignment-check.md
- top-level-structure-check.md

## Validation workflow

Run the repository validation workflow with:

python tools/content-generator/run_repository_validation.py
