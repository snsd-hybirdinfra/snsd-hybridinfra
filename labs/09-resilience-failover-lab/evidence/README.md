# Resilience Failover Lab Evidence

This directory stores lab-local evidence generated during resilience and failover validation execution.

## Evidence Focus

failure detection, failover decision readiness, traffic shift validation, post-failover recovery validation, failback readiness, and resilience evidence.

## Evidence Areas

| Directory | Purpose |
|---|---|
| raw/ | Raw availability checks, failure signals, routing output, load balancer state, monitoring signals, and failover execution records |
| processed/ | Parsed failure detection summaries, failover decision summaries, traffic shift summaries, post-failover recovery summaries, and failback readiness summaries |
| summary/ | Reviewer-readable resilience and failover evidence summaries |

## Current Status

| Area | Status |
|---|---|
| Documentation Status | documentation-ready |
| Implementation Status | planned |
| Execution Status | not yet executed |
| Evidence Status | placeholder until lab execution |

## Evidence Policy

Evidence files in this directory should be generated from actual resilience, failover, traffic shift, and recovery validation once the lab is implemented.

Before implementation, this directory defines the expected evidence boundary only.
