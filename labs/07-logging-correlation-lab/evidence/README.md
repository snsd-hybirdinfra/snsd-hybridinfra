# Logging Correlation Lab Evidence

This directory stores lab-local evidence generated during logging and correlation validation execution.

## Evidence Focus

log collection, event normalization, OpenSearch query readiness, correlation analysis, incident timeline reconstruction, and logging evidence.

## Evidence Areas

| Directory | Purpose |
|---|---|
| raw/ | Raw Linux logs, service logs, container logs, OpenSearch query results, and correlation execution records |
| processed/ | Parsed event summaries, normalized event output, correlation summaries, and incident timeline reconstruction results |
| summary/ | Reviewer-readable logging and correlation evidence summaries |

## Evidence Policy

Evidence files in this directory should be generated from actual log collection, search, and correlation execution once the lab is implemented.

Before implementation, this directory defines the expected evidence boundary only.
