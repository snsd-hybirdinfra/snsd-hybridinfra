# Capability Registry

## Purpose

Defines reusable operational capabilities shared across scenarios, topology models, renderers, and governance workflows.

## Responsibilities

- normalize operational capability naming
- normalize module ownership
- normalize adapter ownership
- standardize capability reuse
- support capability-aware rendering

## Shared Consumers

- scenario-generator
- topology-validator
- relationship-mapper
- diagram-renderer
- governance-checker

## Active Registry

- capability-registry.yaml
