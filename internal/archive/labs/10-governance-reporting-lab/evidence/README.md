# Governance Reporting Lab Evidence

This directory stores lab-local evidence generated during governance reporting validation execution.

## Evidence Focus

repository validation, coverage reporting, evidence summary generation, quality gate validation, documentation consistency, and reviewer-facing governance reporting.

## Evidence Areas

| Directory | Purpose |
|---|---|
| raw/ | Raw repository validation output, markdown link reports, structure check output, coverage data, and report generation records |
| processed/ | Parsed validation summaries, coverage summaries, documentation consistency summaries, and governance evidence summaries |
| summary/ | Reviewer-readable governance and reporting evidence summaries |

## Current Status

| Area | Status |
|---|---|
| Documentation Status | documentation-ready |
| Implementation Status | planned |
| Execution Status | not yet executed |
| Evidence Status | placeholder until lab execution |

## Evidence Policy

Evidence files in this directory should be generated from actual repository validation, coverage generation, report generation, and governance summary execution once the lab is implemented.

Before implementation, this directory defines the expected governance evidence boundary only.
