## Lab Boundary

labs/10-governance-reporting-lab/

The adapter provides the integration boundary between governance validation workflows and repository validation tools, markdown link checks, structure checks, report generation, or Python summary parsing.

## Expected Outputs

- Repository validation result
- Markdown link validation result
- Structure validation result
- README alignment result
- Language validation result
- Coverage summary result
- Governance evidence input data
