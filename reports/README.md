# Generated Reports

This directory contains generated maintainer-oriented reports.

These files support repository maintenance, quality checks, simplification analysis, and generated consistency validation.

## Maintainer Reports

| Report | Purpose |
|---|---|
| portfolio-health-summary.md | Portfolio health summary |
| markdown-link-check.md | Markdown link validation |
| top-level-structure-check.md | Top-level structure validation |
| root-readme-alignment-check.md | Root README alignment validation |
| repository-language-check.md | Repository language policy validation |
| repository-quality-check.md | Repository quality diagnostics |
| repository-simplification-candidates.md | Simplification and cleanup candidates |
| related-scenarios-generation-report.md | Related scenario generation report |
| lab-readiness-alignment-check.md | Lab readiness alignment diagnostics |
| repository-summary-report.txt | Generated repository summary source report |

## Boundary

This directory is primarily for maintainers.

Reviewer-facing documentation belongs under `docs/`.
Reviewer-facing validation summaries belong under `validation-reports/`.
