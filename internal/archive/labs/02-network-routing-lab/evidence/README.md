# Network Routing Lab Evidence

This directory stores lab-local evidence generated during validation execution.

## Evidence Focus

Routing, reachability, DNS, VPN, latency, packet loss, and recovery evidence.

## Evidence Areas

| Directory | Purpose |
|---|---|
| raw/ | Raw command output, telemetry responses, and execution logs |
| processed/ | Parsed validation outputs and normalized summaries |
| summary/ | Reviewer-readable evidence summaries |

## Current Status

| Area | Status |
|---|---|
| Documentation Status | documentation-ready |
| Implementation Status | planned |
| Execution Status | not yet executed |
| Evidence Status | placeholder until lab execution |

## Evidence Policy

Evidence files in this directory should be generated from actual lab execution once the lab is implemented.

Before implementation, this directory defines the expected evidence boundary only.
