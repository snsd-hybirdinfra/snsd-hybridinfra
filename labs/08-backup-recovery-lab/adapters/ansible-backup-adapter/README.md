# Ansible Backup Adapter

## Adapter Purpose

Executes backup, restore, validation, and cleanup automation boundaries.

## Lab Boundary

This is a lab-local adapter for:

labs/08-backup-recovery-lab/

## Integration Role

The adapter provides the integration boundary between backup recovery validation workflows and Ansible automation, filesystem state, service state, integrity checks, monitoring, or evidence parsing.

## Expected Outputs

- Backup job readiness result
- Backup artifact validation result
- Backup metadata result
- Restore workflow result
- Integrity validation result
- Service recovery result
- Evidence input data
