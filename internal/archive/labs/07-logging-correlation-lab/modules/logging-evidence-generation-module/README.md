# Logging Evidence Generation Module

## Module Purpose

Generates raw, processed, and summary evidence from logs and correlation outputs.

## Lab Boundary

This is a lab-local implementation module for:

labs/07-logging-correlation-lab/

## Inputs

- Linux system logs
- Service logs
- Application logs
- Container logs
- OpenSearch query results
- Correlation validation output
- Timeline reconstruction data

## Outputs

- Raw log evidence
- Processed event summaries
- Normalized event summaries
- Correlation analysis summaries
- Scenario evidence summaries
- Lab validation report inputs
