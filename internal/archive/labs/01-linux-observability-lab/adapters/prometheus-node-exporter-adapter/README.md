# Prometheus Node Exporter Adapter

## Adapter Purpose

Collects Linux host metrics through node exporter.

## Lab Boundary

This is a lab-local adapter for:

labs/01-linux-observability-lab/

## Integration Role

The adapter provides the integration boundary between the lab validation workflow and the underlying Linux, monitoring, automation, or evidence collection mechanism.

## Expected Outputs

- Execution output
- Telemetry availability result
- Validation status
- Evidence input data
