# Linux Routing Adapter

## Adapter Purpose

Reads Linux interface, route table, next-hop, and path state.

## Lab Boundary

This is a lab-local adapter for:

labs/02-network-routing-lab/

## Integration Role

The adapter provides the integration boundary between the network routing lab workflow and the underlying routing, telemetry, automation, or validation mechanism.

## Expected Outputs

- Route state output
- Reachability result
- DNS validation result
- Latency and packet loss result
- VPN or tunnel status
- Evidence input data
